# Price Picks 

A smart flight tracking dashboard that lets users search real-time availability and 'pick' their target price. Built to monitor flight price fluctuations and notify the user when prices drop to their picked range.

## Features
- **Real-time Search:** Find current flights between cities.
- **Price Picking:** Set a target budget for specific flights.
- **Watchlist:** Save and monitor your preferred trips.

## Tech Stack
- HTML5, CSS3, JavaScript (ES6+)
